# Sample Images for Vision RAG

This directory should contain sample financial charts and infographics for Vision RAG demonstrations.

## Recommended Sample Images

1. **Financial Charts:**
   - Stock price charts (line, candlestick, bar charts)
   - Revenue/earnings charts
   - Market trend visualizations
   - Economic indicators graphs

2. **Infographics:**
   - Data visualizations
   - Process flow diagrams
   - Comparison charts
   - Statistical infographics

## File Format

- Supported formats: PNG, JPG, JPEG
- Recommended size: 800x600 to 1920x1080 pixels
- File naming: Use descriptive names (e.g., `stock_chart_2024.png`, `revenue_infographic.png`)

## Usage

Sample images can be loaded into chat sessions for Vision RAG demonstrations. Users can ask questions about the visual content, and the system will use Cohere Embed-4 for multimodal search and Gemini 2.5 Flash for visual question answering.

## Adding Sample Images

1. Place image files in this directory
2. Images will be automatically available for Vision RAG queries
3. Use descriptive filenames to help identify content
